Guadalajara Java User Group

Comunidad Java orgullosamente tapatía

Una de las comunidades mas grande de Guadalajara

En Java User Group Guadalajara tendrás oportunidad de conocer otros desarrolladores Java y hablar acerca de código, patrones, innovaciones y compartir tus ideas. Nuestras reuniones son técnicas e incluyen a expertos en la tecnología Java. Damos la bienvenida a principiantes y gurús, a desarrolladores y gerentes, a geeks y profesionales.
